# 🗺️ PokéTracker Roadmap

> Version : 1.0
>
> Ce document décrit la vision à long terme de PokéTracker.
>
> Il ne s'agit pas d'une liste de fonctionnalités.
>
> Il définit les grandes étapes qui permettront à PokéTracker de devenir la meilleure application de gestion de collection Pokémon TCG.

---

# Notre vision

Nous voulons construire une application que les collectionneurs auront plaisir à ouvrir chaque jour.

PokéTracker ne cherche pas à être le site avec le plus de fonctionnalités.

Il cherche à être celui qui offre la meilleure expérience.

Chaque version devra rendre l'application :

- plus agréable
- plus rapide
- plus intelligente
- plus utile

Sans jamais sacrifier sa simplicité.

---

# Les piliers du produit

Toutes les évolutions doivent renforcer au moins un de ces piliers.

---

# Fonctionnalités volontairement exclues

PokéTracker ne deviendra pas :

- un réseau social
- une marketplace
- un système d'enchères
- un jeu
- une plateforme de paris
- un forum

Ces fonctionnalités éloigneraient le produit de sa mission.

---

# Comment prioriser une nouvelle fonctionnalité

Avant d'accepter une idée, répondre à ces questions.

## 1

Améliore-t-elle réellement la vie d'un collectionneur ?

---

## 2

Respecte-t-elle la philosophie du produit ?

---

## 3

Peut-elle être comprise immédiatement ?

---

## 4

Peut-elle être développée sans complexifier inutilement l'application ?

---

## 5

Serons-nous toujours heureux de cette décision dans trois ans ?

Si une réponse est NON,

la fonctionnalité doit être repensée.

---

# Définition d'une version terminée

Une version n'est pas terminée lorsque toutes les fonctionnalités sont développées.

Elle est terminée lorsque :

- le design est cohérent ;
- les performances sont excellentes ;
- les animations sont fluides ;
- les bugs critiques sont corrigés ;
- la documentation est à jour.
